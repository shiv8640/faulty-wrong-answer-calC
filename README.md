# Faulty Calculator 😈

A fun web-based calculator that sometimes gives incorrect results intentionally.

## 🚀 Features
- Basic arithmetic operations
- Random faulty logic (30% wrong answers 😈)
- Clean UI design
- Beginner-friendly project

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 💡 How It Works
- Enter two numbers
- Choose an operation
- Get result (correct OR wrong 😏)
